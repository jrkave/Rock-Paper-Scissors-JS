# Rock Paper Scissors Game
This project implements the game 'Rock Paper Scissors' in JavaScript in two ways. It allows players to play against the computer and keeps track of scores.

# Function-Based vs. Class-Based
This project contains two folders that hold separate versions of the 'Rock Paper Scissors' game. One uses functions to implement the game logic, and the other uses 
classes. However, gameplay is identical in each. These two versions were simply created to explore different ways to achieve the same gameplay. 

# Features
* Play against the computer
* Keep track of player and computer scores
* Display game logic and outcomes

# Installation and Usage

To play, simply clone the repository using `git clone`. Then, open either rockpaperscissors.html or rps.html in a web browser. 
Click on the buttons to make your move (rock, paper, or scissors) and see the outcome of each round.

# Motivation
This project was created to explore JavaScript and learn how to manipulate the DOM.
